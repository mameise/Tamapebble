Tama P1 ROM Converter

Converts a Tamagotchi P1 ROM file to the byte order expected by this emulator.

What this does: The Tamagotchi P1 ROM circulating online uses big-endian byte order, but the tamalib emulator expects little-endian. This tool swaps the byte pairs so the ROM works.

1. Pick your ROM file

Select your rom.bin (or whatever it's called). It should be exactly 12288 bytes.

2. Analyze (optional)

Check the byte order and structure. Tells you if you need to swap or not.

3. Swap bytes and download

Swaps each byte pair and saves a new file tama_p1.bin. This is the file you upload as a resource in CloudPebble.
